はじめに

突然だが、(チュドノフスキー級数で) π を計算したくなった！

Chudnovsky Formula

1 π = 12 ⁢ ∑ n = 0 ∞ ( −1 ) n ⁢ ( 6 n ) ! ( 3 n ) ! ⁢ ( n ! ) 3 ⁢ 13591409 + 545140134 n 640320 3 n + 3 / 2
